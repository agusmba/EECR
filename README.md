# EECR
Web/App del clan Elite España para Clash Royale

### Contenido
SPA con las siguientes secciones:
- Login y Editor de Perfil  
  - Login/Signup  
  - Perfil con nick en el juego, y RRSS
  - Selector de cartas  
- Info del clan (¿comentarios en Disqus?) 
- Ranking de barajas  
  - Top 10 mas votado  
  - Top 10 mas usado  
  - Link ver todo  
    - Lista por páginas, ordenable
